### 基本概念

#### 定义

* 使用`class`关键字定义一个类，后面跟着类名，后面跟着一对花括号，花括号里包含有类的属性与方法的定义。

* 一个类可以包含有**常量**，**属性**，**方法**。

* 表示当前引用对象使用`$this`

  ```php
  // 定义一个SimpleClass类
  class SimpleClass
  {
  	// 声明一个属性
    	public $name = '我是一个属性';
    
    	// 声明一个方法
    	public function method()
      {
        	echo '我是一个方法';
      }
  }
  ```
